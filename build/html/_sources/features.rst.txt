Features
========

- **Interconnected AI Nodes**: A dynamic network where each node can be an AI chatbot or another processing unit.
- **Extendable Architecture**: Easily add new ticket types, queues, languages, and more without significant changes to the system.
- **Random Generators**: Introduce variability in generated outputs using Input/Output Distribution Tables.
- **Cost Efficiency**: Uses smaller, optimized AI models like GPT-4o mini for text rewriting and translation, reducing operational costs.
